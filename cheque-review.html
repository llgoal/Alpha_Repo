Layout Tool

Text placement app

A generic text placement surface inspired by report layout tools. Enter the three fields, then drag and resize each text block on the page.

- Drag any text block to move it.
- Drag the blue handle to resize the text box width/height.
- You can also set exact X/Y and width/height in inches.
- Grid cells are exactly 1 inch, and the canvas is exactly 7.5 in × 3.5 in.
