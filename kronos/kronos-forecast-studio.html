Kronos Forecast Studio

A static control surface for your local Kronos workflow. Use it to generate exact run commands, inspect committed forecast CSVs, and visualize near-term market scenarios for symbols like QQQ and SOFI. It is designed to be Git-friendly: no backend required for the UI itself, while the forecasting engine stays local.
