# Project Organization

Keep self-contained project work grouped under `projects/<project-name>/`.

Current examples:
- `projects/bazi/` — BaZi research, engine, sources, and related UI
- `projects/kronos/` — Kronos runners, UI, and forecast artifacts

Preferred pattern for future additions:
- code
- docs/research notes
- generated sample artifacts that belong to that project
- project-specific UI files

This keeps the repo root reserved for truly top-level items only.
